MEMORIIN (Memory Game)

Janika Aan, Evelin Jõgi

PROJEKTI PLAAN

Mängijaid 1. 4x4 pildikaarti asetatakse lauale, mis on juhuslikult genereeritud programmi poolt.
Mängija valib laual olevatest kaartidest välja kaks kaarti. Kui pildid omavahel sobivad, muutuvad nad mängulaual mitteaktiivseks. Kui kaardid omavahel ei
sobi, pööratakse nad laual ümber (pilt allpool) ja mängija saab valida uuesti kaks kaarti. Mäng lõpeb, kui kõik paarid on laual leitud.
